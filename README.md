# Hubert Brzeg - Lokalne Koło Łowieckie

## Opis Projektu

Strona internetowa dla lokalnego koła łowieckiego "Hubert Brzeg". Projekt zawiera responsywny design dostosowany do urządzeń mobilnych i desktopowych.

## Struktura Projektu

```
hubert-brzeg/
├── index.html           # Główny plik HTML
├── css/
│   └── styles.css       # Arkusz stylów
├── js/
│   └── script.js        # Kod JavaScript
├── images/              # Folder na obrazki
├── assets/              # Folder na inne zasoby
└── README.md            # Ten plik
```

## Funkcjonalności

- **Nawigacja**: Menu nawigacyjne z linkami do różnych sekcji
- **Sekcja Hero**: Powitanie i wezwanie do działania
- **O nas**: Informacje o kole łowieckim
- **Wydarzenia**: Lista nadchodzących spotkań i działań
- **Kontakt**: Informacje kontaktowe

## Technologie

- HTML5
- CSS3 (Responsive Design)
- JavaScript (ES6+)

## Jak zacząć

1. Otwórz plik `index.html` w przeglądarce internetowej
2. Podgląd strony powinien otworzyć się w domyślnej przeglądarce

## Wymagania

- Nowoczesna przeglądarka internetowa (Chrome, Firefox, Safari, Edge)

## Dostosowanie

- Zamień `info@hubertbrzeg.pl` na rzeczywisty adres email
- Zaktualizuj numer telefonu w sekcji Kontakt
- Dodaj rzeczywiste obrazki do folderu `images/`
- Zmodyfikuj zawartość sekcji "O nas" i "Wydarzenia" odpowiadając na potrzeby koła
